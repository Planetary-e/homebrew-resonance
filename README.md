# Homebrew Tap for Planetary Resonance

Privacy-preserving decentralized matching protocol.

## Install

```bash
brew install Planetary-e/resonance/resonance
```

Or tap first:

```bash
brew tap Planetary-e/resonance
brew install resonance
```

## Usage

```bash
resonance
```

Opens http://localhost:3000 in your browser.
